IOCP를 사용한 비동기 C# Server Framework
====

Requirement:
 - .net framework 3.5 이상
 - Winsock version 2.2 이상
 - Visual Studio 10.0 이상

Building
======

Windows
-------------
 core.sln 실행 후 컴파일


/core - 프레임워크가 들어있는 파일

/ServerTest - 웹서버(Port 8888)와 Close()를 하지 않는 서버(Port 8000) 2개 예제

/ClientTest - ServerTest와 짝이되는 서버
